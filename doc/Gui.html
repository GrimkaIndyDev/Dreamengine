Class: Gui

Gui

Gui class is used to create a CanvasGui binded on the given camera. It work like a Scene but Gui is binded to the given Camera and all GameObjects inside will follow the camera's moves. Gui copy the Sizes of the Camera, but at any moment you can disable it or change alpha only for Gui. Warning: CanvasGui isn't always the best choice. Don't forget DOM is more powerful As example, Gui with GameObjects allow you to use dynamics animations, and logics inside your objects like you do to make the game. If you want only simple button with hover and active effect, DOM seems to be a better choice. Try both and choose. so make a CanvasGui if it's easier for you or if you use tricky graphics
